Cấu trúc thư mục

  • app
    • angularjs - Ứng dụng angular
    • commom - Chứa meta data, script, css được nhúng trong file html
    • data - Chứa hình ảnh, dữ liệu demo cho website
    • fonts - Chứa các font được sử dụng trong website
    • images - Chứa hình ảnh của layout
      • generated - Chứa sprites (tăng hiệu suất website)
      • icons - Chứa các icons để tạo sprites
    • modules - Chứa các modulecủa website, dùng include để nhúng vào html
    • scripts
      • libs - Chứa các thư viện không thể cài được bằng bower
        • html5shiv - Tạo các thẻ html5 trên trình duyệt cũ (IE)
        • jwplayer - Hỗ trợ phát video từ youtube, vimeo hoặc video file từ host server
        • respond - Hỗ trợ media query (CSS3) trên trình duyệt cũ (IE)
        • dateformat - Định dạng thời gian, ngày giờ
      • helpers.js - Các hàm tiện ích
      • main.js - Khai báo các đoạn script dùng chung của tất cả các trang
      • settings.js - Cấu hình website
    • styles
      • base - Style các thẻ html, form (core)
        • _base.scss - Style các thẻ html
        • _form.scss - Style các thẻ form
      • layout - Style bố cục chung như header, content, footer...
        • _layout.scss - Style bố cục trang web và những thành phần chung
      • tools - Chứa các variables, mixins, functions, configs hỗ trợ code nhanh hơn
        • _function.scss - Các hàm tiện tích
        • _mixins.scss - Các khối code dựng sẵn
        • _settings.scss - Cấu hình cho css
      • ultils - Chứa các class đã style sẵn, hack theo trình duyệt, giao diện các popup
        • _animation.scss - Hiệu ứng animation CSS3
        • _font.scss - Các font được sử dụng trong website
        • _hack.scss - Hack css cho các trình duyệt
        • _helpers.scss - Các class được style sẵn
        • _popup.scss - Style cho các kiểu popup
    • bower_components - Chứa các thư viện được dùng trong website (đi kèm với bower.json)
    • dist - Thư mục html dev sẽ làm việc sau khi biên dịch
    • node_modules - Các module được sử dụng trong template (đi kèm với package.json)
  • bower.json - Chứa các thư viện được dùng trong website (đi kèm với bower_components)
  • Gruntfile.js - Cấu hình của các module trong thư mục node_modules
  • package.json - Các module được sử dụng trong template (đi kèm với node_modules)

Danh sách các hàm hỗ trợ ():

myPopupVideo(params)

Demo

Cách sử dụng:

    params: {
        title (string): Tiêu đề video,
        videoUrl (string): Tập tin video hoặc url youtube,
        pictureUrl (string): Hình ảnh đại diện cho video,
        width (number - optional): Chiều dài video,
        height (number - optional): Chiều cao video,
        autoPlay (bool - optional): Tự động playvideo,
        playerSkin (string - optional): Giao diện player,
        ratio (string - optional): Tỉ lệ màn hình (16:9, 4:3),
        closeOnBg (bool - optional): Tắt popup khi click overlay
    }
    myPopupVideo({
        title:'Tiêu đề video',
        videoUrl:'https://www.youtube.com/watch?v=Wslh9RTT8CY',
        pictureUrl:'https://i.ytimg.com/vi/Wslh9RTT8CY/mqdefault.jpg'
    });

myPopupAlert(param)

Demo

Cách sử dụng:

    params: {
        title (string): Tiêu đề,
        content (string): Nội dung,
        closeOnBg (bool - optional): Tắt popup khi click overlay,
        redirecTo (string - optional): Chuyển sang trang chỉ định
    }
    myPopupAlert({
        title:'Tiêu đề',
        content:'Nội dung',
        closeOnBg:true,
        redirecTo:'http://google.com'
    });


Lưu ý Nếu muốn redirect sang trang khác thì thêm tham số 'redirecTo' bằng url.

myPopupContent(params)

Demo

Cách sử dụng:

    params: {
        rel (string): Id đối tượng,
        closeOnBg (bool - optional): Tắt popup khi click overlay,
        width (number): chiều rộng popup
    }
    myPopupContent({
        rel:'#popup',
        closeOnBg:true,
        width:800
    });

customRadioCheckbox() | Style cho checkbox và radio

Cách sử dụng:

$('input[type="radio"],input[type="checkbox"]').customRadioCheckbox();

onlyNumber() | Chỉ cho phép nhập số

Cách sử dụng:

$('.onlyNumber').onlyNumber();

Lưu ý Script này đã khai báo sẵn nên chỉ cần thêm class 'onlyNumber' vào input là được.

lockInput(value) | Không cho nhập nội dung vào input, textarea

Cách sử dụng:

$('.lockInput').lockInput(true);

setCookie(name, value, minuteExpires, path, domain, secure) | Tạo cookie

Data

Cách sử dụng:

setCookie('key', 'value', 1);

getCookie(name) | Lấy dữ liệu cookie

Cách sử dụng:

var data = getCookie('key');
alert(data);

deleteCookie(name, path, domain) | Xóa cookie

Cách sử dụng:

deleteCookie('key');

getWindowSize() | Lấy kích thước cửa sổ client

Cách sử dụng:

var wSize = getWindowSize();
// return [x,y]

getDocumentSize() | Lấy kích thước nội dung trang web

Cách sử dụng:

var wSize = getDocumentSize();
// return [x,y]

embedYoutube(link, options) | Nhúng video clip youtube vào website

Cách sử dụng:

$('.ytWrap').append(embedYoutube('https://www.youtube.com/watch?v=HLphrgQFHUQ', { width: 480, height: 320, params: '' }));

limitWords(text, nWords) | Giới hạn từ hiển thị

Cách sử dụng:

var str = limitWords('Lorem ipsum dolor sit amet, consectetur adipisicing elit', 5);
// return 'Lorem ipsum dolor sit amet...'

var key = getQueryString('http://www.domail.com?key=key1&value=value1', 'key');
return 'key1';
